# 📈 Sales Forecasting System

**Sqrock IT Solutions Internship — Data Science Project 1**

A machine learning web app that analyzes historical sales data and predicts future sales using a Random Forest Regressor.

## 🚀 Live Demo
[Click here to open the app](https://your-app-link.streamlit.app)

## 📌 Features
- Upload any sales CSV dataset
- Automatic data cleaning and feature engineering
- Monthly sales trend visualization
- Sales by category chart
- Model training with MAE & RMSE evaluation
- Actual vs Predicted sales comparison chart
- Predict future sales by entering a custom date

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn (Random Forest Regressor)
- Streamlit

## 📂 Dataset
Superstore Sales Dataset — [Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
